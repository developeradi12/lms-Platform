### *# LMS Platform – Full Stack Learning Management System*

## 📄 Description
A full-stack Learning Management System built with Next.js, TypeScript, MongoDB, and TailwindCSS where instructors can create courses and students can enroll, watch lessons, and track their learning progress.


## ⚙️ Requirements
Node.js (v18 or higher), npm or yarn, MongoDB (Atlas or local instance)


## 🎮 Usage
 1.Clone the repository
 2. Install dependencies using npm install
 3. Configure environment variables in a .env file
 4. Start the development server using npm run dev
 5. Open http://localhost:3000 in your browser
